# Breed Improved v0.1.0 Release Checklist

All items remain manual gates until a release owner checks them.

## Repository

- [ ] Working tree reviewed.
- [ ] All intended changes committed.
- [ ] No save files or logs included.
- [ ] GitHub remote is correct.
- [ ] Default branch is correct.
- [ ] Release commit recorded.
- [ ] Tag `v0.1.0` has not yet been created.

## Clean-package test

- [ ] Install from `Breed-Improved-v0.1.0.zip`.
- [ ] Launch with only Breed Improved enabled.
- [ ] Interaction appears.
- [ ] English localisation works.
- [ ] Simplified Chinese localisation works.
- [ ] A free-cost case works.
- [ ] A paid 100-Prestige case works.
- [ ] House Head Hook-present and Hook-absent cases work.
- [ ] Opinion, stress, ten-year modifier, permanent marker, and Dynasty consequences work.
- [ ] Save/reload works.
- [ ] Error log reviewed.

## Steam Workshop

- [ ] Title and description reviewed.
- [ ] Preview image prepared.
- [ ] Compatibility selected.
- [ ] Tags selected.
- [ ] Initial visibility restricted where supported.
- [ ] Uploaded content inspected.
- [ ] Subscribe/download test passed.
- [ ] Workshop version matches the GitHub package.

## GitHub Release

- [ ] Tag `v0.1.0` created from the approved release commit.
- [ ] Clean ZIP uploaded as a Release asset.
- [ ] SHA-256 checksum inserted.
- [ ] Release notes reviewed.
- [ ] Steam Workshop link inserted.
- [ ] Release marked latest.
