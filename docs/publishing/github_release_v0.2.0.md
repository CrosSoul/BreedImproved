# Breed Improved v0.2.0 — GitHub Release Copy Draft

Status: draft only. Runtime acceptance passed; do not publish this GitHub Release without separate authorization.

Breed Improved v0.2.0 adds the player-controlled **Manage Dynasty Cleanup** Decision while preserving the individual **Exile from Dynasty** interaction.

## Changelog

- Added Bloodline and Negative Congenital Trait cleanup modes.
- Added sequential candidate review, spare, early-finish, and add-all-remaining controls.
- Added final confirmation, ancestor-first review, and duplicate branch-root folding.
- Added direct-candidate protection. It prevents direct selection and direct execution, but a protected descendant may still move with the complete descendant branch of a selected ancestor.
- Preserved the existing individual exile cost and consequences.
- Improved English and Simplified Chinese interface text.
- Added no automatic, background, scheduled, or recurring cleanup.

## Compatibility

- CK3: `1.19.*`
- Languages: English and Simplified Chinese

## Installation

Steam Workshop is the only supported ordinary-player installation channel:

https://steamcommunity.com/sharedfiles/filedetails/?id=3769010534

Subscribe, enable **Breed Improved** in a CK3 Launcher playset, and launch the game. No manual-install asset is provided. GitHub-generated source archives are source snapshots, not supported installable Mod packages.
